# SeaTunnel-Web upstream modules (official Apache SeaTunnel Web source).

This directory contains the upstream SeaTunnel Web codebase:

- `seatunnel-server/` — backend
- `seatunnel-ui/` — frontend
- `seatunnel-datasource/` — datasource plugins
- `seatunnel-web-dist/` — distribution packaging
- `seatunnel-web-common/`, `seatunnel-web-it/`, `seatunnel-mpack/`

**Do not add DolphinScheduler integration code here.** All DS/ST integration lives in:

- `integration-server/` — unified backend
- `integration-web/` — unified frontend
- `config/integration.yml` — unified configuration
